# Legacy tests

These files are archived historical tests that target pre-Hardhat-3 or pre-current-contract flows.

They are kept for reference only and are not part of the actively maintained regression suite.

Current actively maintained tests are the Hardhat 3 suite under test-hh3/ and the rewritten source files it imports from test/.

Archived here because they reference one or more removed or obsolete paths:
- SourceDaoToken and related old token flow
- Investment and whitelist investment contracts
- Marketing contract flow
- MultiSig wallet flow
- network-specific attached contract scripts
